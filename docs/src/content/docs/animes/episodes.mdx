## Descripcion

Retorna la lista de episodios precalculados asociados a un anime, con soporte de paginacion por `offset` y `limit`.

## Parametros de ruta

| Parametro | Tipo | Descripcion |
| --- | --- | --- |
| `animeId` | string | Identificador del anime. |

## Parametros de consulta

| Parametro | Tipo | Descripcion |
| --- | --- | --- |
| `offset` | number | Desplazamiento inicial (por defecto `0`). |
| `limit` | number | Maximo de episodios a retornar (por defecto `10`). |

## Ejemplo de uso

```bash
curl -X GET "https://tu-dominio/api/animes/solo-leveling/episodes?offset=0&limit=12"
```

## Respuesta

Cada episodio incluye el `episodeId` que puedes usar en `/episodes/:episodeId` o `/episodes/:episodeId/sources`.
