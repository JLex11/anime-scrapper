## Descripcion

Devuelve las fuentes de reproduccion disponibles para un episodio. El parametro `id` corresponde al `episodeId` obtenido desde los listados de episodios.

El formato de `videos` puede variar segun el proveedor: algunas veces es un arreglo simple y otras un objeto con listas `SUB` o `DUB`. Por eso se recomienda tratarlo como una estructura flexible.

## Parametros de ruta

| Parametro | Tipo | Descripcion |
| --- | --- | --- |
| `id` | string | Identificador del episodio (ejemplo: `solo-leveling-15`). |

## Ejemplo de uso

```bash
curl -X GET "https://tu-dominio/api/episodes/sources/solo-leveling-15"
```

## Ejemplo de respuesta

```json
{
  "episode": 15,
  "videos": [
    {
      "server": "StreamTape",
      "title": "StreamTape",
      "ads": 0,
      "allow_mobile": true,
      "code": "Hk3R1Pq",
      "episodeId": "solo-leveling-15"
    }
  ]
}
```

## Errores comunes

| Codigo | Descripcion |
| --- | --- |
| `404` | No se encontraron fuentes para este episodio. |
| `500` | Error al obtener las fuentes del episodio. |
