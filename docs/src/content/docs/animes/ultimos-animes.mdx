## Descripcion

Devuelve un arreglo con los animes mas recientes visibles en la portada del proveedor. La lista se ordena de mas nuevo a mas antiguo.

## Parametros de consulta

| Parametro | Tipo | Descripcion |
| --- | --- | --- |
| `limit` | number | Cantidad maxima de resultados. Si no se envia, se usa `15`. |

## Ejemplo de uso

```bash
curl -X GET "https://tu-dominio/api/animes/latest?limit=10"
```

## Consideraciones

- Los objetos retornados incluyen la informacion completa del anime (sinopsis, generos, imagenes y fechas).
- Si el proveedor no publica un campo especifico, ese valor puede venir como `null`.
