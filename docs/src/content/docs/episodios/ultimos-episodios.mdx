## Descripción

Obtiene un arreglo con los episodios más recientes detectados en la fuente oficial. La lista siempre se devuelve ordenada de más nuevo a más antiguo.

## Respuesta

Cada elemento del arreglo contiene:

- `originalLink`: URL original del episodio en el proveedor.
- `image`: Ruta optimizada servida desde este API (útil para miniaturas en clientes web).
- `episode`: Número de episodio detectado.
- `title`: Nombre del episodio.
- `episodeId`: Identificador único del episodio, útil para otras rutas.
- `animeId`: Identificador del anime relacionado.

> **Nota:** El playground genera los recursos con la URL base que definas. Si sirves la API detrás de un proxy recuerda ajustar el campo `URL base` para conservar el dominio correcto.
