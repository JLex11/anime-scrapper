## Descripcion

Entrega una lista del feed `rating` precalculado, ordenada por ranking.

## Parametros de consulta

| Parametro | Tipo | Descripcion |
| --- | --- | --- |
| `limit` | number | Cantidad maxima de resultados. Si no se envia, se usa `10`. |

## Ejemplo de uso

```bash
curl -X GET "https://tu-dominio/api/animes/latest/rating?limit=8"
```

## Recomendacion de uso

Si necesitas mostrar un top en tu frontend, combina esta ruta con `/animes/:animeId` para refrescar detalles especificos.
Este endpoint no ejecuta scraping en demanda.
