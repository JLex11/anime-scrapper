## Descripción

Devuelve un arreglo con la información resumida del catálogo. El origen se pagina de manera secuencial y la página inicial es `1`.

## Parámetros de consulta

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| `page` | number | Página solicitada (por defecto `1`). |

## Ejemplo de uso

```bash
curl -X GET "https://tu-dominio/api/animes?page=2"
```

La respuesta es un arreglo de animes que incluye atributos listos para renderizar tarjetas en el frontend.
