## Descripcion

Devuelve el detalle completo del anime, con sinopsis, generos, imagenes y relaciones. Esta ruta lee exclusivamente datos precalculados en Supabase.

## Parametros de ruta

| Parametro | Tipo | Descripcion |
| --- | --- | --- |
| `animeId` | string | Identificador del anime. Se obtiene desde otras rutas. |

## Ejemplo de uso

```bash
curl -X GET "https://tu-dominio/api/animes/solo-leveling"
```

## Respuesta

Incluye `images` con recursos optimizados y, cuando existe, `relatedAnimes` con relaciones semanticas (secuela, spin-off, etc.).

## Errores comunes

- `404`: El anime no existe en la base precalculada.
