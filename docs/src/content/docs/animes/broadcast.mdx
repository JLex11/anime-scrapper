## Descripcion

Retorna una lista del feed `broadcast` precalculado. Esta ruta es util para construir secciones de temporada actual.

## Parametros de consulta

| Parametro | Tipo | Descripcion |
| --- | --- | --- |
| `limit` | number | Cantidad maxima de resultados. Si no se envia, se usa `20`. |

## Ejemplo de uso

```bash
curl -X GET "https://tu-dominio/api/animes/broadcast?limit=12"
```

## Notas

- La respuesta incluye la informacion completa del anime, incluidas URLs finales de imagen.
- Algunos campos pueden variar segun la fuente (por ejemplo `rank` o `status`).
- El endpoint no ejecuta scraping en demanda.
