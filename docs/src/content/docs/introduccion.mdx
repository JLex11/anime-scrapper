## ¿Qué es el Anime Scrapper API?

Esta API facilita el acceso a catálogos de anime, episodios recientes y enlaces de reproducción optimizados. No es necesario registrarse para los ejemplos públicos, pero puedes añadir tus propias claves y cabeceras personalizadas en el playground interactivo.

## Flujo general

1. Consulta la lista de recursos disponibles en el menú lateral.
2. Revisa los parámetros requeridos, ejemplos de respuesta y limitaciones descritas en cada endpoint.
3. Utiliza el botón **Probar endpoint** para lanzar una petición en vivo con tu propia URL base (`https://tu-dominio/api`).

## Siguientes pasos

- Explora los **últimos episodios** para obtener los lanzamientos más recientes.
- Consulta el recurso de **animes** para paginar por temporadas o filtrar por búsquedas.
- Activa la caché del lado del cliente agregando los encabezados requeridos según la documentación de cada ruta.
