## Descripcion

Este endpoint no entrega binario directamente. Devuelve un `302` hacia una URL firmada temporal en Cloudflare R2.

## Como se usa

- La API de animes/episodios ya devuelve campos `image` con este formato:
  - `https://tu-dominio/api/image/<imageToken>`
- Tu frontend solo consume esa URL y sigue la redireccion automaticamente.

## Respuesta esperada

- `302`: token valido, redireccion a URL firmada.
- `404`: token invalido.
- `500`: error al firmar.

## Notas de seguridad

- El `imageToken` no expone el `object key` plano.
- La URL firmada tiene expiracion corta y se genera por request.
