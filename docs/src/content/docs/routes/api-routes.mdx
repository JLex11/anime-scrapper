## Descripcion

Este endpoint lista las rutas expuestas por el servicio con su descripcion y categoria. Es util para construir menus dinamicos o para validar rapidamente el inventario de endpoints disponibles.

## Ejemplo de uso

```bash
curl -X GET "https://tu-dominio/api/api-routes"
```

## Estructura de la respuesta

Cada item del arreglo incluye:

- `route`: URL absoluta construida con el origen actual.
- `description`: resumen corto del endpoint.
- `category`: grupo logico (`animes`, `episodes`, `images`).

## Notas

- La ruta `GET /api/api-routes/docs` redirige a la documentacion interactiva en `/api/api-docs/`.
- Las rutas se generan en tiempo real usando el origen actual (protocolo + host).
