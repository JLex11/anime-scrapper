## Descripcion

Devuelve el episodio que coincida con el `episodeId` como objeto. Si no hay coincidencias se retorna `404`.

## Parametros de ruta

| Parametro | Tipo | Descripcion |
| --- | --- | --- |
| `episodeId` | string | Identificador del episodio (ejemplo: `solo-leveling-15`). |

## Ejemplo de uso

```bash
curl -X GET "https://tu-dominio/api/episodes/solo-leveling-15"
```

## Ejemplo de respuesta

```json
{
  "originalLink": "https://www3.animeflv.net/ver/solo-leveling-15",
  "image": "https://tu-dominio/api/image/ZXhhbXBsZS1lcGlzb2RlLXRva2Vu",
  "episode": 15,
  "title": "Solo Leveling - Episodio 15",
  "episodeId": "solo-leveling-15",
  "animeId": "solo-leveling",
  "created_at": "2026-03-24T10:05:21.832Z",
  "updated_at": "2026-03-24T10:05:21.832Z"
}
```

## Errores comunes

| Codigo | Descripcion |
| --- | --- |
| `404` | Episodio no encontrado. |
| `500` | Error al obtener el episodio. |

> Nota: El campo `image` apunta al endpoint de proxy firmado (`/api/image/:token`).
