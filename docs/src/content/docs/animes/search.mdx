## Descripcion

Permite buscar animes por texto usando el indice interno. La respuesta es un arreglo paginado con informacion parcial para listas rapidas.

## Parametros de ruta

| Parametro | Tipo | Descripcion |
| --- | --- | --- |
| `query` | string | Termino de busqueda. |

## Parametros de consulta

| Parametro | Tipo | Descripcion |
| --- | --- | --- |
| `page` | number | Pagina solicitada (por defecto `1`). |
| `pageSize` | number | Tamano de pagina (por defecto `10`). |

## Ejemplo de uso

```bash
curl -X GET "https://tu-dominio/api/animes/search/solo-leveling?page=1&pageSize=12"
```

## Respuesta

Esta ruta retorna un subconjunto de campos para optimizar el listado:

- `animeId`, `title`, `type`, `status`
- `genres`, `description`, `otherTitles`
- `images` con `coverImage` y `carouselImages`

Si necesitas informacion completa, usa `/animes/:animeId`.
